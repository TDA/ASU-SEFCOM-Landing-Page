SEFCOM - E-Mail Header Injection Detection Project

Arizona State University E-Mail Header Injection Detection Project

This is an academic project to detect if a website is vulnerable to the e-mail header injection attack.

If your website has been crawled by our service, and you want to know whether your website is vulnerable to this attack, do contact us.

Details about the vulnerability

This attack is a pretty old attack that has not received much attention. You can read more about it here: Early 2005 Article about E-Mail header injection
